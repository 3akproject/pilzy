## Overview

**Pilzy** is a smart medicine reminder and health management mobile application built using Flutter.  
It helps users manage their medications, schedules, reminders, and important health documents — all in one place.

Designed as a practical and scalable solution, Pilzy ensures users never miss a dose while keeping their medical data organized and accessible.

---

## Features

### Medication Management
- Add, edit, and delete medicines  
- Store dosage and instructions  
- Organize medicines efficiently  

### Reminders & Notifications
- Schedule medicine reminders  
- Local notifications with alerts  
- Timely reminders to ensure adherence  

### Multi-User Support
- Supports multiple user profiles  
- Separate data management for each user  

### Schedule Management
- Create and manage medication schedules  
- Track daily routines easily  

### History Tracking
- Maintain logs of taken/missed medicines  
- View past medication activity  

### Document Management
- Store medical documents  
- Easy access to prescriptions and reports  

### Local Backup file generation
- User can backup data to local storage
- Can import the backup data in other devices using the file

---

## App Modules

- **Medicines Page** — Manage all medicines  
- **Schedules Page** — Set and track schedules  
- **History Page** — View medication history  
- **Documents Page** — Store health documents  
- **Settings Page** — Manage user and app settings
